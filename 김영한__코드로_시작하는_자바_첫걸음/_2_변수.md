## 변수 시작
### Var1
#### 코드
```java
package variable;

public class Var1 {

    public static void main(String[] args) {
        System.out.println(20); //변경 10 -> 20
        System.out.println(20); //변경 10 -> 20
        System.out.println(20); //변경 10 -> 20
    }
}   
```

#### 실행결과
```zsh
10
10
10
```

#### 설명
* 이번에는 처음으로 패키지(지금은 폴더라고 이해하면 됨)를 만든다
* variable라는 패키지를 만들고 그 안에서 자바 코드를 작성할땐 맨 위에 `package variable;`을 작성해야함.

### Var2
#### 코드
```java
package variable;

public class Var2 {
    public static void main(String[] args) {
        int a; //변수 선언
        a = 20; //10 - > 20

        System.out.println(a);
        System.out.println(a);
        System.out.println(a);
    }
}
```

#### 실행결과
```java
20
20
20
```

### 변수 선언
`int a`
* 위 뜻은 a라는 정수 변수를 선언한다는 뜻이다.

### 변수에 값 대입
```java
a = 10
```
* 자바에서 `=`은 오른쪽에 있는 값을 저장한다는 뜻이다.
* 변수에 처음 값을 대입해서 저장하는 것을 초기화라 한다.

### 변수 값 읽기
* 변수의 값을 쓰려면 그 변수명을 입력하면 된다.
* sout()안에 a를 넣었다면, a에는 10이 저장돼 있기 때문에 10이 들어간 효과가 난다.

## 변수 값 변경
### Var3
#### 코드
```java
package variable;

public class Var3 {

    public static void main(String[] args) {
        int a;//변수 선언
        a = 10; //변수 초기화 a(10)
        System.out.println(a);
        a = 50; //변수 값 변경: a(10 -> 50)
        System.out.println(a);
    }
}
```

#### 실행결과
```zsh
10
50
```

#### 설명
컴퓨터는 위에서 아래로 읽기 때문에 10이 먼저 출력되고, 50이 출력 된다.

## 변수 선언과 초기화
### 변수 선언
변수는 다음과 같이 하나씩 선언하거나
```java
int a;
int b;
```

한번에 선언할 수 있다.
```java
int c,d;
```

### 변수 초기화
다음 처럼 여러 초기화 방법이 존재한다.

```java
int a;
a = 10;

int b = 2;

int c = 3, d = 4;
```

변수는 초기화하지 않으면 컴파일에러가 생길 수 있다.

## 변수 타입1
### Var7
#### 코드
```java
package variable;

public class Var7 {

    public static void main(String[] args) {
        int a = 100; //정수
        double b = 10.5; //실수
        boolean c = true; //불리언(boolean) true, false 입력 가능
        char d = 'A'; //문자 하나
        String e = "Hello Java"; //문자열, 문자열을 다루기 위한 특별한 타입

        System.out.println(a);
        System.out.println(b);
        System.out.println(c);
        System.out.println(d);
        System.out.println(e);
    }
}
```

#### 실행결과
```zsh
```
```
100
10.5
true
A
Hello Java
```

#### 설명

* int : 정수
* double : 실수
* boolean : 불리언
* char : 문자 하나 (작은 따음표 사용)
* String : 문자열. (큰 따음표 사용)

## 변수 타입2
### Var8
#### 코드
```java
package variable;

public class Var8 {

    public static void main(String[] args) {
        //정수
        byte b = 127; //-128 ~ 127
        short s = 32767; // -32,768 ~ 32,767
        int i = 2147483647; //-2,147,483,648 ~ 2,147,483,647 (약 20억)

        //-9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807
        long l = 9223372036854775807L;

        //실수
        float f = 10.0f;
        double d = 10.0;

    }
}
```

#### 설명
* long 사용시 정수리터럴 뒤에 `L`을 붙여야 한다.
* 실수 리터럴은 기본이 `double`이다. 만약 `float`형을 사용한다면, 뒤에 `f`를 붙여야 한다.

---

## 변수타입 정리
### 실무에서 사용하지 않는 타입
* float : 표현길이와 정밀도가 낮아, `double`을 사용
* char : 문자 하나만 쓰는 경우는 많이 없다. 문자가 하나여도 `String`을 쓰는 편.

---

## 변수 명명 규칙
클래스는 대문자로시작하고, 나머지는 소문자로 시작.

### 낙타 표기법
여러 단어로 이루어진 변수이름의 경우, 첫번째 단어는 소문자로 시작하고 그 이후는 대문자로 시작.
